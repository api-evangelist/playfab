Microsoft released an open-source Xbox Godot sample on GitHub to help developers build games for Xbox on PC using the Godot engine. The sample demonstrates integration of the Microsoft GDK, Xbox Services, and PlayFab, covering features like multiplayer functionality, game saves, and controller support. The initiative aims to lower barriers for Godot developers interested in publishing to the Xbox platform.
